# 🎮 鲸天 Minecraft 服务器入服指南

> 一份简单易懂的新手入服教程，跟着步骤走，几分钟即可进入服务器！

---

## 第一步：启动游戏

打开 **PCL CE 启动器**，推荐版本为 **1.21.11**，点击「启动游戏」。

![启动游戏](https://cnb.cool/Snowflake-2025/DrawingBed/-/git/raw/main/myblog/jingtian-server-guide/1.jpg)

---

## 第二步：进入多人游戏

在 Minecraft 主菜单中，选择 **「多人游戏」**。

![进入多人游戏](https://cnb.cool/Snowflake-2025/DrawingBed/-/git/raw/main/myblog/jingtian-server-guide/2.jpg)

弹出第三方服务器警告提示后，直接点击 **「继续」** 即可。

![点击继续](https://cnb.cool/Snowflake-2025/DrawingBed/-/git/raw/main/myblog/jingtian-server-guide/3.jpg)

---

## 第三步：添加服务器

在多人游戏界面，点击下方的 **「添加服务器」** 按钮。

![添加服务器](https://cnb.cool/Snowflake-2025/DrawingBed/-/git/raw/main/myblog/jingtian-server-guide/4.jpg)

填写服务器信息：

| 选项 | 填写内容 |
|------|----------|
| **服务器名称** | 随意起一个你喜欢的名字 |
| **服务器地址** | 电信：`mc.jingtian888.shop`<br>移动：`pa.jingtian888.shop` |

> 💡 博主使用的是移动地址，大家可以根据自己的网络情况选择。

![填写服务器信息](https://cnb.cool/Snowflake-2025/DrawingBed/-/git/raw/main/myblog/jingtian-server-guide/5.jpg)

填写完成后点击 **「完成」**。

---

## 第四步：加入服务器

在服务器列表中，选中你刚添加的服务器，然后点击 **「加入服务器」**。

![选择服务器](https://cnb.cool/Snowflake-2025/DrawingBed/-/git/raw/main/myblog/jingtian-server-guide/6.jpg)

![加入服务器](https://cnb.cool/Snowflake-2025/DrawingBed/-/git/raw/main/myblog/jingtian-server-guide/7.jpg)

---

## 第五步：注册账号

首次进入服务器后，需要在聊天框输入注册命令：

```
/register <密码> <确认密码>
```

例如：`register 123456 123456`

![注册账号](https://cnb.cool/Snowflake-2025/DrawingBed/-/git/raw/main/myblog/jingtian-server-guide/8.jpg)

> ⚠️ 请牢记你的密码，下次登录需要使用！

---

## 第六步：登录（再次进入时）

以后每次进入服务器，需要在聊天框输入登录命令：

```
/login <密码>
```

![登录账号](https://cnb.cool/Snowflake-2025/DrawingBed/-/git/raw/main/myblog/jingtian-server-guide/9.jpg)

登录成功后，就能愉快地玩耍了！🎉

![成功进入](https://cnb.cool/Snowflake-2025/DrawingBed/-/git/raw/main/myblog/jingtian-server-guide/10.png)

---

## 常用命令一览

| 命令 | 功能 |
|------|------|
| `/sb` | 关闭面板 |
| `/shop` | 打开商店 |
| `/res` | 领地相关 |
| `/warp` | 地标传送 |
| `/tpa` | 传送到其他玩家 |
